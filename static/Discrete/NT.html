1.2 Divisibility Relation
Deﬁnition 1.1. b|a (read ”b divides a” or ”b is a divisor of a” or ”a is a multiple of b”) is deﬁned by
b|a ⇐⇒ a = bx for some x ∈ Z
This is equivalent (if b is positive) to saying the remainder is zero independent of the signs of a and b. From
this deﬁnition, we can say 0|0. If 0|a, then a must be zero—there is no integer x such that 0x = a 6= 0. A
few things to note:
1. The divides relation is reﬂexive: a|a∀a ∈ Z, with q = 1.
2. The divides relation is transitive: if a|b and b|c, then a|c ∀a, b, c ∈ Z. Basically, we multiply by
something, and then something else.
3. If d|a, then d|(a · x)∀x ∈ Z.
4. If d|a and d|b, then d|(a + b)
5. It follows 3. and 4. that if d|a and d|b, then d|(ax + by)∀x, y ∈ Z. This can be read as ”d divides any
integer linear combination of a and b.”
1.3 Primes
The number 1 can be thought of as the ”basic building block of positive integers by addition.” I.e., if we
have any positive integer x, we can construct x by adding suﬃciently many values of 1 together. In a similar
sense, primes are ”the building blocks of positive integers by multiplication!”
Deﬁnition 1.2. Prime: a number a ≥ 2 is said to be prime if the only positive integers that divide a are
1 and a. Note that we say here that we specify a ≥ 2, so 1 is not a prime. In a moment, we’ll see that this
is to make the Fundamental Theorem of Arithmetic cleaner.

Lemma 1.2. For every integer a ≥ 2, we have p|a for some prime p.
Proof. (By contradiction): Suppose, to obtain a contradiction, that Lemma 1.2 is false, and let a ≥ 2 be the
smallest integer not divisible by any prime.
The number a cannot itself be prime, because if it were, it would divide itself (because the division re-
lation is reﬂexive), and would thus be divisible by a prime.
Then a must be composite. Let a = bc with b ≥ 2 and c ≥ 2. Then b is smaller than a, so p|b for
some prime p (this is because we deﬁned a as the SMALLEST integer for which the Lemma fails, so any
integer smaller than a must satisfy the Lemma). But then, since b|a, we have p|a, a contradiction. So, there
does not exist a smallest counterexample to the Lemma, so it must hold for all a ≥ 2. 

1.4 Remarks
• Divisibility is Reﬂexive and Transitive.
• However, it’s not quite antisymmetric; for instance −1|1 and 1|−1, but −1 6= 1.
• Something that’s reﬂexive and transitive is called a pre-order. If we add in antisymmetric, we get an
order.
• If (a|b ∧b|a), we deﬁne this as a ≡ b. Note, this is an equivalence relation (i.e., it is reﬂexive, transitive,
and symmetric).
• Equivalence classes partition a ”universe of discourse” into sets of elements related by an equivalence
relation. Associates are members of the same equivalence classes.

3 The Euclidean Algorithm and Modular Arithmetic
3.1 Euclid’s Algorithm
There is no eﬃcient algorithm for factorization of numbers into primes. In fact, it’s a good thing there
aren’t—essentially all encryption for sending things like credit card information over the internet to an
